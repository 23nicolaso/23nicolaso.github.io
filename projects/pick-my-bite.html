Pick My Bite

Mobile app that helps groups decide where to eat.

Overview

Pick My Bite is a full-stack mobile application that simplifies group dining decisions by aggregating preferences and nearby restaurant data.

Architecture

  • React Native frontend
  • Serverless Node.js backend on AWS Lambda
  • PostgreSQL database for caching and persistence
  • Google Places API integration

Engineering Decisions

  • Implemented API response caching to reduce Google API costs
  • Designed stateless Lambdas for horizontal scalability
  • Normalized database schema to avoid duplicate place records

What I Learned

  • Designing cost-aware cloud architectures
  • Managing async flows in serverless environments
  • Building production-ready mobile APIs
